## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rech99/Portafolio.git
cd Portafolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📝 Customization

### Update Your Information

Edit `app/page.tsx` and replace:
- "Your Name" with your actual name
- Job title and descriptions
- Experience entries with your career history
- Skills with your technologies
- Projects with your portfolio items
- Contact links and social media

### Modify Colors

Colors are defined using Tailwind CSS utility classes. Main colors used:
- **Primary**: `bg-blue-600` / `hover:bg-blue-700`
- **Background**: `bg-white` / `dark:bg-slate-950`
- **Text**: `text-gray-900` / `dark:text-white`

To change the color scheme, find and replace color classes in `app/page.tsx`.

## 🚀 Deployment

### Deploy to Vercel (Recommended)

The easiest way to deploy is with [Vercel](https://vercel.com/):

1. Push your code to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your GitHub repository
4. Click "Deploy"
5. Your portfolio will be live in minutes!

**Environment Variables**: No additional setup required for this project.
